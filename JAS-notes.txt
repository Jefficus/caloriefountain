Building locally
    - edit conf-dev.py if nec.
    - nikola clean; nikola build --conf=conf-dev.py

Validation
    - linkchecker --no-status --no-warnings http://localhost:8000

Deploy
    - diff conf-dev.py conf.py   # be sure no important diffs
    - nikola build -a; nikola github_deploy

    # Note: that 'build -a' ensures that everything gets built - ignores caching
    # Maybe not required for pushing new posts but definitely required
    # when updating templates and other site structures/code
